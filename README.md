# newjob
Search your new job.

## Precondition
```sh
virtualenv -p python3 venv
source venv/bin/activate
pip install -r requirements.txt
```

## Run debug mode
```sh
python debug.py
```

## Run production mode
```sh
python run.py
```

## Run job spider
```sh
python job_spider.py
```

## Run Tests
```sh
nose2 -v
```
